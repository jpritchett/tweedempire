# Tweed Project Full Pack

This package contains the latest Godot 4.x prototype plus the earlier archive zips.

## Latest project
`tweed_project/godot/` is the full project ready to open in Godot (open `project.godot`).

## Archives
`tweed_project/archives/` contains the earlier zip snapshots for reference.

## Controls (latest)
- Left click: select nearest entity
- Right click: move selected unit using A* over hidden grid
- B: toggle build mode
- 1/2/3: choose structure to place (patchbay / loomcore / irrigation)
- Left click (build mode): place structure
